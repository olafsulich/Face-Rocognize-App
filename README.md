## Code Example/Issues 🔍

I used Face API documentation when I had some problems.

```bash
 const detections = await faceapi
      .detectAllFaces(image)
      .withFaceLandmarks()
      .withFaceDescriptors();
    const resizedDetections = faceapi.resizeResults(detections, displaySize);
```
